About MedRAG

Architecture & technology stack.

🔍
Retrieval

🧠
Embeddings
HuggingFace all-MiniLM-L6-v2 Sentence-Transformer — 384-dim dense vectors.
✍️
Generation
Google flan-t5-base — instruction-tuned seq2seq model, runs locally without API keys.
🌐
Backend
Flask REST API with lazy-loaded RAG engine, streaming-ready architecture.
📦
Data
Auto-detects disease–symptom CSVs and plain-text files. No preprocessing required.
⚡
Pipeline
Query → Embed → FAISS Search → Prompt Construction → LLM → Structured Response.
